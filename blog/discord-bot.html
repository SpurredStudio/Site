Discord Bot

The Discord Bot is a visual studio solution made out of almost entirely C#. Though the Discord Bot was used as a learning tool so not everything is as well programmed as it could be. That being said the Discord Bot is programmed with the idea of private use in mind and not for use on multiple servers.

Here is a list of the features available in the Discord Bot.

  • Music - Users can request songs right off youtube
  • Imgur - get random photos from imgur subreddits
  • Vote Ban - can vote to ban non Mod players
  • Clean Chat - Mods can delete messages from chat
  • Fun Commands - Commands like 8ball, roll, and raffle
  • Admin Only Commands - Commands used to manage server
  • Clean Chat - Mods can delete messages from chat
  • Grant Channel - Mods can grant other users channels
  • Admin Text Channel - Shows who was using what admin command
  • Twitter Feeds - Post recent tweets from a given twitter account every 60 seconds. At the moment to change twitter feeds you have to edit the code.

Though The bot is tailored to our Discord server so it will not work right out of the box. However once you have replaced a few lines of code to tailor the bot to your server it should run just fine. Specifically login information for the bot and twitter feed would need to be changed, and a couple lines that store channel and role information.
